**✨ Features**

**Module and Features****
**Dashboard**-Summary cards (balance, income, expenses, savings rate) • 6-month bar chart • Spending donut chart • Recent transactions |
**Transactions**-Add / Edit / Delete • Search • Filter (type, category, month) • Sort any column • CSV export |
**Insights**-Top spending category • Savings rate gauge • Monthly comparison • Smart observations |
**UX**-Dark/Light theme • Admin/Viewer roles • Keyboard shortcuts • Toast notifications • Fully responsive |

**Usage**

Action-How to us
Add transaction-Click "+ Add Transaction" (Admin mode)
Edit/Delete-Click ✏️ or 🗑️ icons
Filter-Search box or dropdowns
Sort-Click any column header
Export-Click "Export" button
Switch role-Sidebar dropdown
Toggle theme-Click 🌙/☀️ icon
Keyboard shortcuts-Ctrl+N (New), Ctrl+D (Dashboard), Ctrl+T (Transactions), Ctrl+I (Insights)
---
**🔧 Customization**

Add category – Edit CATS array in script.js
Change currency – Modify fmt() function in script.js
Update colors – Edit CSS variables in style.css
